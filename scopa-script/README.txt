== Description ==

== Installation ==

= Automatic Installation =

1. Go to **Plugins -> Add New**.
2. Search for **Scopa Script**.
3. Install and activate the Scopa Script plugin.

= Manual Installation =

1. Download the latest tagged archive.
2. Go to **Plugins -> Add New** 
3. Click on **Upload Plugin**.
4. Upload the zipped archive directly.
5. Activate the Scopa Script plugin.
